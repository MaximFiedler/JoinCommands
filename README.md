## Events
- Player join
- Player first join
- Player quit server
- Player death
- Player switch world

## Default config
```
Commands:
  Player:
 - replace these lines
  - with your commands
  Console:
 - replace these lines
  - with your commands
```

The commands can be executed by the console or the player  
You can use the **%player%** placeholder in the commands

## Command
> **Note**
> Need the `joincommands.reload` permission
/joincommands-reload
